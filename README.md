### Spaces in Space

**Space in Space 개요:**

**'Spaces in Space' 는 ‘공간 안에 공간들’이라는 컨셉으로 1인 가구를 위한 고급스러우면서 가성비 좋은 맞춤형 가구를 선보이는 온라인 쇼핑몰이며, 작은 공간에서도 세련된 분위기와 실용성을 제공합니다.**

**사이트 주요 기능:**

- 가구 종류별로 편리하게 조회
- 회원 가입 후 장바구니 추가, 구매하기
- 1:1 문의하기
- 상품별 리뷰 작성
- FAQ(공지) 조회
